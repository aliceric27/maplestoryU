NXPC 申領頁面開放公告

公告 2025年04月14日, 12:56 (UTC+8)

為了表彰早期參與楓之谷宇宙發展的貢獻者，NEXPACE 將向廣泛的早期貢獻者分發 14,222,857 NXPC（佔 Avalanche C-chain 總供應量的 1.422%）。這些貢獻者包括 Open Beta Test 玩家、預發行活動用戶、任務完成者、社群成員和錯誤回報者——他們都在實現 MSU 願景的過程中扮演了至關重要的角色。

申領期從2025年5月15日開始，至2025年8月15日結束。
符合資格的楓之谷玩家可以透過官方申領頁面查看其狀態並申領 NXPC。

有關資格詳情和申領說明，請參閱以下內容。

■ 如何查詢資格

  1. 訪問 NXPC 申領頁面 (可於 2025年5月15日, 14:00 (UTC+8) 起訪問)

    • 前往 https://msu.io/claim (請確保網址正確)

  2. 連接您的錢包

    • 連接您用於參與相關活動的 *MetaMask 錢包。
    • 確保您的錢包已連接到 Avalanche C-Chain 網路。

⚠️注意

  • 如果您透過第三方服務（例如 Kaito）使用非 MetaMask 錢包參與相關活動，請將相應的私鑰匯入 MetaMask，以確保順利檢查資格。
  • 為此，您可以參考 MetaMask 關於使用私鑰匯入帳戶的官方指南以獲取逐步說明。

  1. 註冊並接受條款 (僅限新用戶)

    • 如果您尚未擁有 MSU 帳戶，請先註冊以繼續。
    • 仔細閱讀並接受服務條款、最終用戶許可協議、隱私政策以及任何相關政策，以便繼續申領。

  2. 檢查資格和可申領金額

    • 連接錢包後，申領頁面將顯示您的資格狀態以及您有權申領的 NXPC 金額。

■ 如何申領

請完成上述步驟 1–4 以檢查您的資格。

然後按照以下步驟申領您的 NXPC：

  1. 同意申領條件

    • 如果您符合資格，請點擊「申領」按鈕。
    • 您需要勾選相應的方塊，確認您同意 NXPC 申領條款和條件並知曉其他必要事項，然後才能繼續。

  2. 簽署交易

    • 查看交易詳情並在您的 MetaMask 錢包中簽署申領交易。
    • 確保您有足夠的 *AVAX 來支付 Avalanche C-Chain 上的 Gas 費用。
    • 請在錢包提示時確認交易。

⚠️注意

  • 申領交易在 Avalanche C-Chain 上執行，需要一定數量的 AVAX（Avalanche 的原生代幣）來支付 Gas 費用。
  • 如果您的錢包中沒有足夠的 AVAX，則無法完成申領。Gas 費用由網路決定，並可能因各種因素而波動——這些因素超出我們的控制範圍，並可能隨時更改，恕不另行通知。
  • 請注意，在發起申領之前，您需要在錢包中存入足夠數量的 AVAX。

  1. 確認申領成功

    • 交易提交後，請等待區塊鏈上的確認。
    • 申領完成後，網站上將顯示成功訊息。
    • 然後，檢查您的錢包以確認已收到 NXPC 代幣。

※ 如果您沒有自動看到代幣，您可能需要手動將 NXPC (在 Avalanche C-chain 上) 添加到您的錢包中。

  1. 打開 MetaMask，滾動到資產列表底部，然後點擊「匯入代幣」。

  2. 然後，輸入以下代幣合約地址：

    • NXPC 合約 (Avalanche C-Chain): 0x5E0E90E268BC247Cc850c789A0DB0d5c7621fb59

  3. 有關逐步說明，請參閱 MetaMask 官方指南：如何新增自訂代幣 – MetaMask 支援

■ 後續步驟

成功申領 NXPC 後，您可以在整個楓之谷宇宙中使用它：遊戲、物品交易、應用程式功能或參與遊戲內市場。

  1. 橋接到 Henesys L1

    • 使用 MSU Warp 將 NXPC 從 Avalanche C-Chain 橋接到 Henesys L1。
    • 橋接後，您的 NXPC 將成為 Henesys 上的原生實用代幣，並作為解鎖楓之谷宇宙生態系統中無限實用功能的關鍵。
    • 它將用於即將推出的遊戲，如楓之谷N和其他楓之谷宇宙協同應用程式。

  2. 在支援的交易所上存取 NXPC

    • NXPC 將在各種支援的交易所（中心化交易所 CEX 和去中心化交易所 DEX）上提供，可用於生態系統內的實用功能交易。
    • 請注意，不同平台的存款和提款網路可能有所不同，因此在支援的交易所上進行任何活動之前，請務必仔細檢查所選網路。

⚠️注意

  • NXPC 將可在 Binance Alpha（透過 Binance App 和網頁版）、Binance Wallet 以及基於 BSC 的 DEX（如 PancakeSwap）上交易。(查看幣安公告)
  • 要在這些平台上交易 NXPC，您需要使用專用的跨鏈路徑將您申領的 NXPC 從 Avalanche C-Chain 橋接到 BNB Chain。(連結)
  • 其他中心化交易所支援透過 Avalanche C-Chain 進行 NXPC 的存款和提款。如果您計劃將申領的 NXPC 轉移到您的 CEX 帳戶，請在繼續操作前確保存款地址使用「AVAX C-Chain」網路。

更多使用 NXPC 的方式將在不久的將來公佈。
敬請期待更新。

■ 01. Genesis Point

Genesis Quest 活動是楓之谷宇宙首個長期貢獻和參與計劃。
該活動從2024年6月24日持續到2025年3月4日，透過與 Galxe 整合的任務進行，作為我們預發行生態系統的基礎。
參與者通過完成各種反映對專案實際投入的任務來賺取 Genesis Point。這些積分代表了楓之谷玩家在開發早期階段的進展和支持，當時 NXPC 尚未具有公開的代幣價值。

  • 獎勵標準
    Genesis Point 獎勵涵蓋了廣泛的生態系統參與——從簡單的社交任務到更積極的任務參與，包括參與 Open Beta Test。

    • 在第二次 Pioneer Test 和 Gear Up 期間完成的任務被賦予額外權重，反映了它們在實際測試和更深入參與中的重要性。
    • 這些任務被視為 Genesis 活動的主要任務線，額外的 Genesis Point 會自動授予——無需額外申領。
    • 根據遊戲行為和錢包數據應用了內部女巫過濾，以確保公平性。

  • 資格注意事項

    • 僅計算截至2025年3月4日 04:59 UTC (即 2025年3月4日 12:59 UTC+8) 之前申領的任務。未申領的積分之後無法恢復或要求。
    • 被標記為違反 MSU 服務條款的錢包，包括帳戶交易或從受限地區訪問，將被取消獲得任何獎勵的資格。
    • 最終的 NXPC 金額是根據總 Genesis Point、活動參與質量和內部評分演算法的組合計算的。每個符合條件的錢包都會收到個人化的分配，而非統一分配。

Genesis Point 的分配是對那些在我們早期旅程中一直陪伴我們、持續參與楓之谷宇宙願景並幫助塑造我們未來旅程基礎的人們的致敬。

■ 02. NXPC 競賽

  • NXPC 競賽於2024年11月20日至29日在第二次 Pioneer Test 期間舉行，設有強化和市場兩個類別。
  • 參與者根據其 NESO 代幣使用情況爭奪最高排名，活動期間實時排行榜追踪活動情況。
  • 總計 2,013,610 NXPC 分配給了兩個類別的頂尖表現者。
  • 獎勵分配反映了最終排名，包括決勝局調整。被識別為多帳戶或濫用行為的楓之谷玩家被排除在獎勵之外。 (查看公告)

■ 03. 楓之谷N 7日簽到

  • 在第二次 Pioneer Test 期間總共完成7天楓之谷N每日簽到的參與者將獲得 10 NXPC 作為獎勵。 (查看公告)

■ 04. Bug Bounty

  • 在參與者中，分享了寶貴反饋的「20名隨機選中的探索者」和「5名主要問題識別者」獲得了 NXPC 獎勵。 (查看公告)

■ 05. MSU Alphabet Fusion 活動

  • 在2025年3月17日至3月24日舉行的「Alphabet Fusion」活動期間，參與者合作提交 Alphabet、Number 或 Name Length NFT 來完成一個39格的收集板——每提交一個 NFT 可獲得 1 NXPC。
  • 總共銷毀了 15,747 個 NFT，並根據個人貢獻向參與者分配了等量的 NXPC。 (查看公告)

■ 06. Kaito：Connect 最高投票者

  • 為了識別和獎勵在 X (Twitter) 上活躍的 Web3 社群貢獻者，我們與 AI 驅動的 InfoFi 網路 Kaito 進行了整合。
  • 為了將 MSU 專用的 Yapper 排行榜整合到 Kaito 中，楓之谷宇宙首先必須在 Kaito Connect 上贏得 Yapper Launchpad。
  • 楓之谷宇宙取得了創紀錄的勝利，在當時獲得了 Kaito 歷史上最高的票數——這一切都在短短兩天內完成。
  • 前500名投票者是根據當時 Kaito 官方機制計算的總投票權選出的，該機制對 Yapper Votes 和 Holder Votes 的權重相同 (50:50)。
  • 所有500名選定的參與者都獲得了相等的 NXPC 獎勵。

※ 要獲得獎勵資格，參與者在投票時還需要將其主要的 EVM 錢包連接到 Kaito Connect。

■ 07. Kaito：MSU 排行榜 (Pre-TGE)

  • 在2024年5月1日至2025年4月30日期間，透過 Kaito 上的 MSU 社群排行榜衡量的 X (Twitter) 上 MSU 相關討論的前2,000名貢獻者被選中獲得 NXPC 獎勵。
  • 這不僅包括高影響力的帳戶，還包括在此期間持續發布高質量 MSU 相關內容的較小創作者。
  • 參與者根據其相對貢獻水平被分類到獎勵等級，每個等級獲得固定數量的 NXPC。
  • 在快照期之後，X 上的貢獻將繼續被追踪，包括 TGE 後的活動，並可能被考慮用於未來的獎勵計劃。敬請期待更新！

※ 與楓之谷宇宙團隊相關的帳戶被排除在獎勵分配之外。

※ 要獲得獎勵資格，參與者在快照時還需要將其主要的 EVM 錢包連接到 Kaito Yaps。

■ 早期參與獎勵

除了 NXPC 分配外，楓之谷宇宙還為早期社群參與者提供了一系列以實用為導向的獎勵，以表彰他們的時間、努力和有意義的參與——超越代幣本身。
這些獎勵旨在表彰在專案形成階段的真正參與，並鼓勵在生態系統內和鏈上的持續參與。

■ Make Special Name

「Make Special Name」NFT 系列，由 Alphabet、Number 和 Digit NFT 組成，在第二次 Pioneer Test 和 Gear Up 期間免費分發並可供參與者申領。
這些 NFT 允許楓之谷玩家在正式推出前創建並保留獨特的遊戲內暱稱。
透過組合特定類型，楓之谷玩家可以將其名稱鑄造為 Reserved Name NFT，從而確保早期個性化並在楓之谷宇宙生態系統中建立獨特身份。
由於社群對確保有意義和令人難忘的名稱組合的強烈興趣，「Make Special Name」NFT 在 OpenSea 等公開市場上積極交易，楓之谷玩家在那裡獲取名稱組件以個性化和保留遊戲內名稱。
隨後，一些 Reserved Name NFT——特別是那些具有稀有或象徵性名稱的 NFT——也在市場上交易，反映了社群的創造性表達和早期參與。
「Make Special Name」NFT 系列成為迄今為止 Avalanche 上交易最活躍的 NFT 系列之一，完全由有機興趣和社群參與推動。
重要的是，楓之谷宇宙沒有以任何方式從此過程中獲利——沒有收取版稅，整個分發過程都是透過免費申領進行的。

截至2025年5月 (來源：OpenSea, Dune)

  • Make Special Name NFT

    • 透過免費申領鑄造了 241,158 個 NFT
    • 總交易量超過 11.4K+ AVAX (約 $395K+)
    • 在 Avalanche NFT 歷史總交易量中排名第二

  • Reserved Name NFT

    • 鑄造了 18,823 個獨特的 Reserved Name NFT
    • 最高記錄售價：200 AVAX (約 $5K+)

■ LIVE World Leap

作為其早期社群獎勵計劃的一部分，楓之谷宇宙在第二次 Pioneer Test 和 Gear Up 期間推出了獨家的 LIVE World Leap 計畫。
此活動允許符合資格的楓之谷玩家——那些完成了預先分配任務的玩家——將測試網階段的部分角色和物品進度轉移到正式的楓之谷N上線服務中。
符合資格的參與者被授予其測試網角色、裝備和寵物的鑄造權，允許他們以 NFT 的形式將這些轉移到正式環境中。
這種轉移是透過 NXPC Fission 協議實現的，該協議利用 NXPC 生成符合轉移資格的 NFT。這些 NFT 可以在正式服務開始後重新鑄造以在遊戲中使用。
重要的是，這項福利完全是透過遊戲內參與和進展獲得的，強化了該計劃的社群優先和實用驅動的性質。

※ 部分 NXPC 分配已指定用於 NXPC Fission 協議，以便透過 LIVE World Leap 計劃向符合資格的早期社群成員分發 NFT。
