NXPC 申領頁面開放公告

公告 2025年04月14日, 12:56 (UTC+8)

為了感謝大家早期一起參與楓之谷宇宙的發展，NEXPACE 這次要發放 14,222,857 顆 NXPC（佔 Avalanche C-chain 總供應量的 1.422%）給各種早鳥貢獻者。只要你是 Open Beta Test 玩家、預發行活動用戶、任務完成者、社群成員或有回報過 bug，都有機會拿到！你們都是 MSU 這個願景能走到今天的關鍵人物。

申領時間是從 2025/5/15 到 2025/8/15。
只要你有資格，就可以到官方申領頁面查詢並領取 NXPC。

資格細節和怎麼領，下面都幫你整理好了！

■ 怎麼查詢自己有沒有資格？

  1. 進入 NXPC 申領頁面（2025/5/15 14:00 (UTC+8) 開放）

    • 直接去 https://msu.io/claim（網址要認明喔）

  2. 連接你的錢包

    • 用你參加活動時用的 MetaMask 錢包連接。
    • 記得切到 Avalanche C-Chain 網路。

⚠️注意

  • 如果你之前是用第三方服務（像 Kaito）不是用 MetaMask 參加活動，請把那個錢包的私鑰匯入 MetaMask，這樣才能順利查資格。
  • 不會匯入？可以參考 MetaMask 的官方教學：如何用私鑰匯入帳戶，有詳細步驟。

  1. 註冊並同意條款（新用戶才需要）

    • 還沒有 MSU 帳號的話，先註冊一個。
    • 記得把服務條款、最終用戶協議、隱私政策等都看過並同意，才能繼續領取。

  2. 查詢資格和可領 NXPC 數量

    • 錢包連上後，申領頁面會直接顯示你有沒有資格、可以領多少 NXPC。

■ 怎麼領 NXPC？

上面 1～4 步驟都做完，確認有資格後，接下來就可以領 NXPC 囉：

  1. 同意申領條件

    • 有資格的話，點「申領」按鈕。
    • 要勾選同意 NXPC 申領條款和相關事項，才能繼續。

  2. 簽署交易

    • 看一下交易細節，然後在你的 MetaMask 錢包裡簽署申領交易。
    • 記得錢包裡要有足夠的 AVAX 來付 Avalanche C-Chain 的 Gas 費。
    • 錢包跳出提示時，按確認就好。

⚠️注意

  • 申領是在 Avalanche C-Chain 上進行的，所以一定要有 AVAX（Avalanche 的原生幣）來付 Gas。
  • 沒 AVAX 就沒辦法完成申領。Gas 費用會隨網路狀況變動，這部分我們沒辦法控制，請多留意。
  • 記得在申領前，先把足夠的 AVAX 充進錢包。

  1. 確認申領成功

    • 交易送出後，等區塊鏈確認一下。
    • 申領成功後，網站會顯示成功訊息。
    • 再去錢包裡看看 NXPC 代幣有沒有到帳。

※ 如果錢包沒自動顯示 NXPC，記得手動加一下（Avalanche C-chain 上的 NXPC）。

  1. 打開 MetaMask，滑到資產列表最下面，點「匯入代幣」。

  2. 輸入這個代幣合約地址：

    • NXPC 合約 (Avalanche C-Chain): 0x5E0E90E268BC247Cc850c789A0DB0d5c7621fb59

  3. 詳細步驟可以看 MetaMask 官方教學：如何新增自訂代幣 – MetaMask 支援

■ 領完 NXPC 可以做什麼？

領到 NXPC 後，你可以在楓之谷宇宙裡用它：不管是遊戲、物品交易、App 功能還是遊戲內市場都用得到！

  1. 橋接到 Henesys L1

    • 用 MSU Warp 把 NXPC 從 Avalanche C-Chain 橋到 Henesys L1。
    • 橋過去後，NXPC 就會變成 Henesys 上的原生代幣，是解鎖宇宙生態系統各種功能的關鍵。
    • 未來像楓之谷N或其他宇宙 App 都會用到它。

  2. 在支援的交易所存取 NXPC

    • NXPC 會在各大支援的交易所（CEX、DEX）上架，可以用來交易或生態系統內消費。
    • 不同平台的存提款網路可能不一樣，操作前記得再三確認。

⚠️注意

  • NXPC 之後可以在 Binance Alpha（App/網頁）、Binance Wallet、BSC 上的 DEX（像 PancakeSwap）交易。(看幣安公告)
  • 要在這些平台交易 NXPC，記得要用專用的跨鏈橋把你領到的 NXPC 從 Avalanche C-Chain 橋到 BNB Chain。(橋接連結)
  • 其他 CEX 也支援用 Avalanche C-Chain 存提 NXPC。如果你要轉到 CEX，記得存款地址一定要選「AVAX C-Chain」網路。

未來 NXPC 的更多用法會陸續公開，敬請期待！

■ 01. Genesis Point

Genesis Quest 是楓之谷宇宙第一個長期貢獻活動。
從 2024/6/24 到 2025/3/4，大家只要在 Galxe 上完成 MSU 任務就能拿到 Genesis Point，這些積分代表你在早期就支持我們，當時 NXPC 還沒公開價值呢！

  • 獎勵怎麼算？
    Genesis Point 涵蓋各種參與，從簡單社交任務到深度參與（像 Open Beta Test）。

    • 第二次 Pioneer Test 和 Gear Up 期間的任務會有加權，因為這些是實際測試、參與度高。
    • 這些主要任務線的 Genesis Point 會自動發放，不用另外申請。
    • 我們有用 "sybil" 內部過濾程式，根據遊戲行為和錢包資料審核，確保公平。

  • 資格注意事項

    • 只算 2025/3/4 04:59 UTC（台灣時間 12:59）前申領的任務，沒領的積分之後不能補。
    • 違反 MSU 服務條款的錢包（像帳戶交易、受限地區訪問）會被取消資格。
    • 最終 NXPC 數量會根據總 Genesis Point、參與質量和內部演算法綜合計算，每個錢包分配都不一樣。

這波 Genesis Point，就是要感謝一路陪我們走來、幫忙打造楓之谷宇宙的大家！

■ Make Special Name

「Make Special Name」NFT 系列（包含 Alphabet、Number、Digit NFT），在第二次 Pioneer Test 和 Gear Up 期間免費發放，大家都可以來領。
這些 NFT 讓你在正式開服前就能創建、保留專屬遊戲暱稱。
只要組合特定類型，還能鑄造成 Reserved Name NFT，早早確保個人化、建立獨特身份。
因為大家都很想搶有意義又好記的名字，「Make Special Name」NFT 在 OpenSea 等市場超熱絡，大家都在那邊收集組件來拼自己想要的名字。
後來有些 Reserved Name NFT（特別是稀有或有象徵意義的）也在市場上交易，展現社群的創意和早期參與。
這系列 NFT 目前是 Avalanche 上交易最活躍的 NFT 之一，完全靠社群熱情推動。
重點是，楓之谷宇宙完全沒從這過程賺錢——沒收版稅，全部都是免費申領！

■ LIVE World Leap

早期社群獎勵還有一個重點：「LIVE World Leap」計畫，在第二次 Pioneer Test 和 Gear Up 期間推出。
只要你有完成指定任務，就能把測試網時期的部分角色和物品進度轉移到正式的楓之谷N服務。
有資格的玩家會拿到角色、裝備、寵物的鑄造權，可以把這些東西變成 NFT，轉到正式環境。
這個轉移是靠 NXPC Fission 協議實現的，用 NXPC 生成符合資格的 NFT，正式開服後就能再鑄造回遊戲用。
這些福利都是靠遊戲內參與和進度拿到的，完全社群優先、實用導向！

※ 部分 NXPC 配給已經指定給 NXPC Fission 協議，讓 LIVE World Leap 計畫能發 NFT 給早期社群成員。
