楓之谷 N 官方上線倒數網站開放！

公告 2025.05.01, 13:00 (UTC+8)

楓之谷 N 倒數計時網站現在開放囉！快來設定楓之谷 N 正式上線的提醒吧！

■ 倒數網站開放時間

2025 年 5 月 1 日 13:00 (UTC+8)

■ 倒數網站內容介紹

倒數網站提供了關於即將推出的楓之谷 N 的關鍵細節，以及各種重要資訊。

[前往倒數網站]

  1. 楓之谷 N 正式上線倒數計時

    • 你可以查看距離楓之谷 N 上線還有多少天。

    • 遊戲將在倒數計時結束的那一刻正式上線。

    ※ 倒數網站將隨著楓之谷 N 的上線而關閉。

  2. 預先下載

    • 預先下載期間：2025 年 5 月 8 日 13:00 (UTC+8) ~ 直到楓之谷 N 正式上線

    • 在楓之谷 N 正式上線之前，你可以直接從倒數網站預先下載遊戲客戶端。

    ※ 關於楓之谷 N 預先下載的詳細資訊將會進一步公佈。
    ※ 要預先下載遊戲客戶端，你需要一個已連接到楓之谷宇宙網站的錢包帳戶。關於錢包創建和連接的說明將在不久的將來公佈。

  3. 上線紀念禮物

    • 為了慶祝楓之谷 N 的上線，將會舉辦各種特別活動。

    • 除了下表中顯示的硬幣騎寵和開幕派對套裝外，還有其他獨家活動獎勵等著你。敬請期待，不要錯過！

    • 關於活動和獎勵的詳細資訊將透過未來的活動公告提供。

  4. Discord 特殊身份組

    • 在倒數期間進入 楓之谷宇宙官方 Discord 的玩家將獲得獨家 Discord 身份組。
      ※ 倒數期間將隨著楓之谷 N 的正式上線而結束。

    • 前往官方 Discord 的 #community-news 頻道，並獲得 @Orange Mushroom 身份組。

  5. 楓之谷 N 指南

    • 你可以使用網頁右側的楓之谷 N 指南部分，在 楓之谷 N 官方 Gitbook 中找到各職業的詳細指南。

    • 關於楓之谷 N 的深入資訊將在官方 Gitbook 上更新 - 敬請關注！

請繼續支持楓之谷 N。
